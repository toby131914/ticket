安裝擴充套件

只需安裝一次，之後直接使用搶票頁面即可

1

下載擴充套件資料夾

從 GitHub 下載整個 repository，找到 extension/ 資料夾並記住位置。

2

開啟 Chrome 擴充功能頁面

在 Chrome 網址列輸入：chrome://extensions，按 Enter。

3

開啟開發人員模式

右上角切換 開發人員模式 為開啟。

4

載入未封裝項目

點擊左上角 載入未封裝項目，選擇剛才的 extension/ 資料夾。

5

複製擴充套件 ID

安裝後在擴充功能頁面找到「搶票助手」，複製下方的 ID（一串英文字），填入 index.html 的 EXT_ID 變數中，再 push 到 GitHub。

⚠️ 每台電腦的擴充套件 ID 都不同。如果要讓多人使用，建議將擴充套件發布到 Chrome Web Store，屆時 ID 會固定，不需要各自修改。
